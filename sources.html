Figma

Figma is a collaborative interface design tool that facilitates real-time design collaboration and allows developers to gather design specifications, generate assets, and prototype interactions for seamless integration with code.

VS Code

VS Code is a versatile code editor with features like syntax highlighting, code completion, and debugging, making coding more efficient for developers.

GitHub

GitHub Platform for version control and code collaboration using Git.

Netlify Hosting and deployment platform for web projects with features like continuous deployment and serverless functions.

freeCodeCamp is a nonprofit organization dedicated to providing free coding education to millions of learners worldwide. Its comprehensive curriculum covers web development, data science, and more, offering hands-on projects and certifications to help learners build practical skills and kickstart their careers in tech.

Codecademy is a popular online platform offering interactive coding courses covering a wide range of programming languages and technologies. From HTML and CSS basics to advanced JavaScript frameworks, Codecademy provides learners with a structured curriculum and hands-on projects to enhance their coding skills.

Frontend Mentor provides real-world coding challenges designed to improve frontend skills and build professional portfolios.
